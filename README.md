# Verma Consultancy Services — Official Website

Professional Tax, Audit & Advisory website for **Verma Consultancy Services**, Lucknow, UP.

## ✅ 100% Free Deployment — GitHub Pages

**No server. No API key. No payment. Ever.**

### Deploy in 3 Steps

**Step 1 — Create GitHub account (free)**
→ Go to https://github.com and sign up

**Step 2 — Create a new repository**
→ Click **New repository**
→ Name it: `verma-consultancy` (or anything)
→ Set it to **Public**
→ Click **Create repository**

**Step 3 — Upload files**
→ Click **uploading an existing file**
→ Drag ALL files from this ZIP into the upload area
→ Click **Commit changes**

**Step 4 — Enable GitHub Pages**
→ Go to your repo → **Settings** → **Pages**
→ Under **Source** → select **main** branch → **/ (root)**
→ Click **Save**

**Your website is live in ~1 minute at:**
`https://YOUR-USERNAME.github.io/verma-consultancy`

## How Real-Time Updates Work (Free)

The updates section uses **free public RSS feeds** from:
- TaxScan.in (covers CBIC, CBDT, MCA, EPFO daily)
- CA Club India (daily tax compliance news)
- Converted via rss2json.com (free, no account needed)

**What happens on every page open:**
1. Pre-baked April 2026 data shows instantly (no blank screen ever)
2. Live RSS feeds fetched in background (free)
3. If live feeds load → replaced with today's real news + green "Live" badge
4. If feeds unavailable → pre-baked data stays (always reliable)

**Zero cost. Zero server. Zero API key. Works forever.**
